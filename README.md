# HackBio-Internship-2026_AI-in-Genomics-1
The main contents of this internship include the following:
- Python for AI: Mastering core Python libraries for biological data analysis and AI workflows.
- Machine Learning in Bio (Py): Designing, evaluating, and interpreting machine learning models for biological prediction.
- Deep Learning in Genomics 1: Building and applying convolutional neural networks (CNNs) for identifying genomic patterns and classifying medical images.
